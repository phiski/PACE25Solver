# Heuristiken für das Dominating Set Problem

Dieses Repository enthält die Implementierung meiner Bachelorarbeit im Fachbereich Informatik. Das Projekt befasst sich mit der Entwicklung und Evaluation von effizienten Heuristiken zur Lösung des (NP-schweren) Dominating Set Problems in Graphen.

## 🚀 Kernfunktionen
- **Algorithmen:** Implementierung verschiedener Heuristiken (z.B. Greedy-Ansätze) zur Bestimmung möglichst minimaler dominierender Mengen.
- **Datenstrukturen:** Effiziente Darstellung von Graphen in Java.
- **Performance:** Optimierung der Laufzeit für die Verarbeitung größerer Instanzen.

## 🛠 Technologien & Tools
- **Sprache:** Java (JDK 21)
- **Build-Tool:** [Maven]
- **Versionierung:** Git

## 📈 Hintergrund der Arbeit
Das Dominating Set Problem ist ein klassisches Problem der Graphentheorie mit Anwendungen in Netzwerkdesign, Routing und Datenanalyse. Da das Problem NP-schwer ist, liegt der Fokus dieser Arbeit auf der praktischen Anwendbarkeit durch approximative Lösungsverfahren.

## 💻 Installation & Ausführung
1. Repository klonen:
   ```bash
   git clone https://github.com
   ```
2. Kompilieren und Starten:
   javac Main.java *.gr (Zur Ausführung wird eine *.gr-Datei benötigt, welche den Graphen beschreibt)
